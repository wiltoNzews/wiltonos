# PsiOS Starter

This isn't an app. It's a mirror.

## What It Does

When you talk to an AI with this, something is different.
The AI responds like it knows something about you.
Because it does — you gave it a moment that mattered.

## Quick Start (No Code)

1. Open `PROMPT.md`
2. Copy everything below the line
3. Paste into ChatGPT, Claude, or any AI
4. Talk

The AI will ask about a moment that changed you.
Share one. Then ask it anything.
Notice the difference.

## Deeper Experience (With Code)

```bash
python psi.py
```

Follow the prompts. Breathe. Remember. Talk.

**Requirements:** Python 3.7+ and either:
- Ollama running locally (default), or
- An OpenRouter API key in `~/.openrouter_key`

## What's Actually Happening

When you share a moment, it becomes context.
The AI weaves that moment into responses.
Add more moments, more weaving.

Some people call these "crystals" — condensed memories.
Some track them with symbols.
You don't have to. The basic experience works without any of that.

## Commands

```bash
python psi.py              # First time: guided breath + memory + talk
python psi.py talk         # Continue talking with your field
python psi.py add          # Add another memory
python psi.py list         # See your memories
```
